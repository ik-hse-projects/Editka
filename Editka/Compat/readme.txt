В этой директории лежат файлы, которые позволяют проекту, разработанному под .net framework, работать под .NET 5.
- Framework.cs требуется при компиляции под .net framework
- Compat.cs при компиляции под .NET 5
Файлы проектов автоматически делают так, чтобы всё компилировалось правильно и без ошибок. 

Эти файлы не содержат ничего нового, всю необходимую документацию можно легко найти на docs.microsoft.com.
Поэтому я не стал их документировать, буду надеяться, что этого файла будет достаточно.
